# WearWorth v1

A Next.js full-stack starter for an identity-led fashion brand.

## Run
1. Install Node.js LTS.
2. Open this folder in VS Code.
3. Run `npm install`.
4. Run `npm run dev`.
5. Open `http://localhost:3000`.

## Included
- Original WearWorth visual direction and philosophy
- Responsive homepage, catalogue, product page, cart and about page
- Local cart persistence
- Products API route
- MongoDB/Prisma production schema foundation
- Uploaded WearWorth logo
